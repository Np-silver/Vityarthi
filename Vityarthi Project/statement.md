Problem Statement: Real estate buyers and sellers often struggle to determine the fair market value of a property due to fluctuating market trends and complex variables (location, size, amenities).

Scope of the Project: This project will utilize historical housing data to build a Machine Learning regression model. It includes a web-based user interface for inputting property details and visualizing price trends.


Target Users: Home buyers, real estate agents, and property sellers.

High-level Features:

User-friendly input form for property details (BHK, Square Footage, Location).

Real-time price prediction using a trained ML algorithm.

Interactive data visualizations showing price distribution and correlations.
